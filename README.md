# Acteedog - Official Releases

A local-first desktop app that unifies your work logs for AI

## 📥 Download

Download the latest version from [Releases](https://github.com/ymtdzzz/acteedog-release/releases/latest).

### Supported Platforms

- **macOS** - Apple Silicon (aarch64)
- **Windows** - x86_64
- **Linux** - x86_64

## 🔄 Auto-Update

If you already have Acteedog installed:

1. Open the app
2. Go to **Settings** > **About**
3. Click **"Check for Updates"**
4. Restart the app
